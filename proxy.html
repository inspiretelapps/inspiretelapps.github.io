How to Use:

  1. For your dashboard: Use this URL as your proxy: https://your-username.github.io/proxy.html
  2. Target URL: The full Yeastar API endpoint you want to call
  3. Method: Usually POST for authentication, GET for data
  4. Body: JSON payload (for POST requests)

Example for Authentication:

  • Target URL: https://meyerattorneys.pbx.ycmcloud.co.za/openapi/v1.0/get_token
  • Method: POST
  • Body: {"username": "your_client_id", "password": "your_client_secret"}

Troubleshooting:

  • If one proxy service doesn't work, try another from the dropdown
  • Check the browser console for detailed error messages
  • Some proxy services have rate limits or may be temporarily down
